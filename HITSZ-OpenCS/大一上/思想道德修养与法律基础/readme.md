## 思修

本课程为考查课，占 2 学分。（至少 20 级还是考查课）

分数构成 =10% 平时分+20% 小组展示分 + 20% 期中论文分 + 50% 期末考试

### 关于教师

  PPT风格正常。

### 关于考试

题型有单选题、多选题、辨析题、材料分析题、小作文题。

我们老师最后一节课会划重点，根据重点背几个小时就 ok 了（可以拿很高分），勇者可以裸考。

考试难度，背了不难，不背很难。
